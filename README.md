🔗 # 🎮 Simon Says Game

A fun and interactive memory game built using **HTML, CSS, and JavaScript**. The game challenges players to remember and repeat an increasingly long sequence of colors. As the level increases, the sequence becomes more difficult, testing your memory and concentration.

## 🚀 Live Demo

🔗 https://khanimran001.github.io/SimonSays/

## 📂 GitHub Repository

🔗 https://github.com/khanimran001/SimonSays

## ✨ Features

- 🎯 Random color sequence generation
- 🧠 Memory-based gameplay
- 📈 Level progression
- 🏆 High Score saved using Local Storage
- ✨ Button flash animations
- ❌ Game Over screen with restart option

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🎮 How to Play

1. Press any key to start the game.
2. Watch the color that flashes.
3. Repeat the sequence by clicking the buttons.
4. With every level, a new color is added to the sequence.
5. If you click the wrong button, the game ends.
6. Try to beat your highest score!
